Project use python3. 

There are three files for PRank:

1.prank_data_clean.py
include split function, feature generate function for training data and test data, function for converting data to matrix

2.prank.py
include prank function to train data and get theta and b, predict function, and functions to compute MAE, RMSE, specificity, sensitivity, precision, accuracy
(we compute precision and accuracy, but they are not included in our report)

3.main.py
include main function for running and output. L can be changed.
